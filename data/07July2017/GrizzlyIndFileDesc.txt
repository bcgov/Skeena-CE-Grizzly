
File format for Grizzly indicator file

--------------------------------------------
GrizzlyInd.txt

Strata fields:
      Year: current or projected year
      Scn: e.g. CurrCond
      strata (unique index, or unit, for following static strata):
         TroutUnit
         Landform 
         GBPU
         Ecosection
         LU
         BEC

Output fields:
      aTot: area (ha) in stratum

      // land use type area (ha)
      aLU_NaturalPermanent: area of permanent natural land-use (e.g. lakes, glaciers, parks)
      aLU_Range: area of range land use (e.g. ranches, but not the large areas in the north)
      aLU_Agricultre: area of agricultural and rural land use
      aLU_Natural: area of natural land-use (e.g. working forest)
      aLU_Urban: area of urban land use
      aLU_Industrial: area of industrial land use

      aPark: area of protected areas and conservancies

      // Pipelines
      kmPipelines_existing: length (km) of existing pipelines or assumed in model scenario
      kmPipelines_proposed: length (km) of all proposed pipelines

      // Mines
      nMines_Aggregate: number of existing aggregate mines
      nMines_Coal: number of existing coal mines
      nMines_Mineral: number of existing mineral mines (acid and non-acid generating)
      nMines_Future: area (ha) of future mine sites (not number of mines; modelled; no type specified)

      // Wind Farms
      nWindFarms: number of future wind farms (proposed and/or modelled)

      // Transmission lines
      // "potential" lines connect existing transmission lines to wind energy sites
      kmTransLine_existing: length (km) of existing or projected transmission lines
      kmTransLine_potential: length (km) of potential future transmission lines

      // Forest condition
      aProductiveForest: area of productive forest
      aConif: area of productive coniferous forest
      aMidSeralConif: area of productive coniferous mid-seral (40-80 or 40-120 depending on BEC)

      // Roads
      kmRoad: length (km) of roads

      // Road Density area (ha)  with 0 km/km2 (class 1), 0.1 to 0.3 km/km2 (class 2), 
      // 0.31 to 0.6 km/km2 (class 3), 0.61 to 0.75 km/km2 (class 4) or above 0.75 km/km2 (class 5)
      aRdDensityCls_1
      aRdDensityCls_2
      aRdDensityCls_3
      aRdDensityCls_4
      aRdDensityCls_5

      // Relative human pressure index
      // Class 1: 0 < relative pressure <= 1,000 (remote)
      // Class 2: 1,000 < relative pressure <= 5,000 (low-moderate)
      // Class 3: 5,000 < relative pressure <= 10,000 (moderate-high)
      // Class 4: 10,000 < relative pressure <= 20,000 (high)
      // Class 5: relative pressure > 20,000 (very high)
      aHumanPressureCls_1	
      aHumanPressureCls_2
      aHumanPressureCls_3
      aHumanPressureCls_4
      aHumanPressureCls_5

      // Security class area (ha)
      aSecureFull: area of habitat that has full security 
      aSecurePartial: area of habitat that has partial security
      aSecureNonHab: area of non-habitat that has security
      aNonSecureHab: area of habitat that is not secure (due to roads or patch size)
      aNonSecureNonHab: area of non-habitat that does not have security

     // Grizzly BEI class
     aBEIcls_1: area (ha) of BEI class 1
     aBEIcls_2: area (ha) of BEI class 2
     aBEIcls_3: area (ha) of BEI class 3
     aBEIcls_4: area (ha) of BEI class 4
     aBEIcls_5: area (ha) of BEI class 5
     aBEIcls_6: area (ha) of BEI class 6
     aBEIcls_8: area (ha) of BEI class 8 (fresh water)
